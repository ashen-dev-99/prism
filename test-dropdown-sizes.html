Dropdowns automatically adjust their size based on screen size:

Screen Size Breakpoints:

Desktop (xl+): Uses the selected size (sm, md, lg, xl, xxl)

Large (lg): Automatically switches to medium size

Medium (md): Automatically switches to small size

Small (sm): Uses full width with small padding
